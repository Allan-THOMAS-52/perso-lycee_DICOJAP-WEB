アルファベット Main 動詞 (どうし) 単語　(たんご)



Particule の :

Indique la possession, avec le format : possesseur の possédé :

Exemple : ゼルダの猫。 --> Le chat de zelda

Particule と :

Indique l'énumération -> et . Se place entre deux noms :

Exemple : 　マリオとルイージ --> Mario et Luigi

On peut aussi l'utiliser pour dire avec

Exemple : 　マリオと --> Avec Mario

Particule や :

Indique l'énumération non-exhaustive :

Exemple : 　マリオやルイージ --> Mario et Luigi, etc
ou Entres autres Mario et Luigi

Ne s'applique pas aux verbes. On peut en mettre beaucoup.

Particule は :

Particule du thème de la phrase -> nom ou groupe nominal qui est toujours en début de phrase,

qui annonce ce dont la phrase parle; (≠ sujet du verbe ):

Exemple : 　マリオは男です。 --> Mario est un Homme

Particule に　:

Indique le lieu dans lequel il ne se passe rien -> verbes de présence ( いる = pour ce qui est vivant/animé;　ある = non vivant/inanimé ) :

Exemples:　un homme: いる / un stylo: ある .

Exemples avec に　: 　ホテルにいる。 = être à l'hôtel;

　ナルトはレストランにいる。 = Naruto est au restaurant;

　私はサクラとレストランにいる。 = Je suis au restaurant avec sakura;

Particule で　:

Indique le lieu dans lequel il se passe qqchose:

Exemples:　 ナルトはレストランで食べる。 = Naruto mange au restaurant.

ヒナタは銀行でサクラとテンテンと話す。 = Hinata parle à sakura et tenten à la banque.

Indique le moyen ( ex = transport ... )

Exemples : 私はタキシーで行く。 = je vais en taxi.

ナルトはスプーンで食べる。 = Naruto mange avec une cuillère --> au moyen d'une cuillère.
